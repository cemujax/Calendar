## 🚀 Quick start

1. Clone the repository
   ```sh
   $ git clone https://github.com/cemujax/Calendar.git
   ```

2. Install dependencies
   ```sh
   $ cd Calendar/
   $ yarn
   ```

3. Run the development server
   ```sh
   $ yarn start
   # open localhost:3000
   ```
